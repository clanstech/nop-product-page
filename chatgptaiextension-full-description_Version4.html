How the Plugin Works

The ChatGPTAIExtension Plugin works through a simple and structured configuration process inside the nopCommerce admin panel.

  1. Enable the Plugin Go to the Configure page and enable using the "Enable AI-Powered Services" checkbox.
  2. Select Provider & Enter API Key Choose your AI provider (Azure OpenAI, ChatGPT, DeepSeek, Gemini, or AnythingLLM) and enter the required API Key. Save the settings.
  3. Enable Content Sections Enable checkboxes for Product / Blog / News / Topic Description, Meta Title, Meta Keywords, and Meta Description generation.
  4. Click "Generate Using AI" On any Product, Blog, News, or Topic page, click the button. The Customize Query popup opens where you enter language, keywords, word limit, tone, and special instructions.
  5. AI Processes & Returns Content Your entered values are sent as tokens along with the configured prompt to the selected AI provider. It returns a high-quality, SEO-optimized description instantly.
  6. Meta Tags Generated Automatically Enabling Meta fields allows automatic generation for all four content types. Meta generation works after the page is saved and contains sufficient description content.
