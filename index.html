The ChatGPTAIExtension Plugin for nopCommerce is a powerful AI-driven content generation solution supporting 5 AI models — Azure OpenAI, ChatGPT, Gemini, DeepSeek, and the free AI option AnythingLLM. It automatically creates SEO-friendly descriptions for Products, Blog posts, News Items, and Topic pages. The plugin also generates Meta Title, Meta Keywords, and Meta Description based on configuration settings.

Designed to save time and reduce manual effort, it ensures consistent and professional content across your website. With one-click functionality and secure API integration, it helps improve search engine visibility and overall store performance.

Whether you manage a small store or a large-scale eCommerce platform, this plugin simplifies content creation, improves productivity, and enhances overall store performance across nopCommerce 4.6, 4.7, 4.8, and 4.9.

Configuration

After installing the plugin, navigate to Configuration → ChatGPTAI Extension in your nopCommerce admin panel. The configuration page lets you set up everything in one place — enable AI services, select your preferred AI provider, enter your API key, toggle content sections (Products, Blog, News, Topics, Meta Tags), and customize the default AI prompts for each content type.

Once configured, the "Generate Using AI" button appears on the relevant editor pages throughout your admin panel. Simply click it, customize your query in the popup, and let AI generate professional, SEO-optimized content instantly.
