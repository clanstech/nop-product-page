The ChatGPTAIExtension Plugin works through a simple and structured configuration process inside the nopCommerce admin panel.

  1. Enable the Plugin Go to the Configure page and enable the plugin using the "Enable AI-Powered Services" checkbox.
  2. Select Provider & API Key Select the provider (Azure OpenAI, ChatGPT, DeepSeek, Gemini, or AnythingLLM) and enter the required API Key. Save the settings.
  3. Enable Content Sections Enable specific checkboxes: Allow to generate Product / Blog / News / Topic Description, Meta Title, Meta Keywords, Meta Description.
  4. Generate Content Click "Generate Using AI" on any page to open the Customize Query popup. Enter language, keywords, word limit, tone, and instructions.
  5. AI Processes & Returns Content Values are sent with the configured prompt to the selected AI provider. It returns a high-quality, SEO-optimized description based on your inputs.
  6. Meta Tags Generated Automatically Enabling Meta fields in configuration allows automatic generation for all four content types. Meta generation works after the page is saved with sufficient content.
