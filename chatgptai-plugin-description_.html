Configuration

After installing the plugin, navigate to Configuration → ChatGPTAI Extension in your nopCommerce admin panel. The configuration page lets you set up everything in one place — enable AI services, select your preferred AI provider, enter your API key, toggle content sections (Products, Blog, News, Topics, Meta Tags), and customize the default AI prompts for each content type.

Once configured, the "Generate Using AI" button appears on the relevant editor pages throughout your admin panel. Simply click it, customize your query in the popup, and let AI generate professional, SEO-optimized content instantly.

Customize AI Query Popup

When you click "Generate Using AI", a popup opens where you can customize:

— Name (auto-filled, editable)
— Language Selection
— Key Features & Keywords
— Word Limit
— Tone of Voice / Custom Tone
— Special Instructions

All values combine with the configured prompt and are sent to the selected AI provider. Content is generated in the selected language.
