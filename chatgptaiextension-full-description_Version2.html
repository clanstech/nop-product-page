📖 Overview

The ChatGPTAI Extension Plugin by Clanstech brings the power of artificial intelligence directly into your nopCommerce admin panel. Built on Microsoft Azure OpenAI (ChatGPT), it enables store administrators to generate both short and long product descriptions instantly — eliminating the tedious, time-consuming task of writing unique content for every product in your catalog.

Whether you manage a small boutique or a large-scale e-commerce operation with thousands of products, this plugin dramatically accelerates your content creation workflow while maintaining consistent quality and brand voice across your entire store.

🔧 How It Works

  1. Install the Plugin Download the ChatGPTAI Extension plugin and install it via the nopCommerce admin panel (Configuration → Plugins → Local Plugins).
  2. Configure Azure OpenAI Credentials Enter your Microsoft Azure OpenAI API credentials in the plugin settings to connect the AI engine to your store.
  3. Generate Descriptions Navigate to any product edit page and click the "Fill Using ChatGPT" button to generate a short or long description instantly.
  4. Review & Publish Review the AI-generated content, make any desired edits, and save. Your SEO-optimized description is now live!
